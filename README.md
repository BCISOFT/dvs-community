> **Bienvenue !** Les contributions en français sont les bienvenues. N'hésitez pas à ouvrir des issues ou participer aux discussions dans la langue de votre choix.

## What is DevSpaces?

DevSpaces is a CLI tool that simplifies Docker-based development environments. It provides instant, reproducible development setups for PHP developers (WordPress, PrestaShop, Laravel, Symfony, and more).

**Key features:**
- One-command environment setup
- Pre-configured recipes for popular frameworks
- Automatic SSL certificates for local development
- Database management and backups
- Cross-platform support (macOS, Linux)

## How to Use This Repository

### Report a Bug

Found a bug? Please [open an issue](../../issues/new?template=bug_report.yml) with:
- DevSpaces version (`dvs version`)
- Your OS and Docker version
- Steps to reproduce
- Expected vs actual behavior

### Request a Feature

Have an idea? [Open a feature request](../../issues/new?template=feature_request.yml) and describe:
- The problem you're trying to solve
- Your proposed solution
- Any alternatives you've considered

### Ask a Question

For questions and general discussions, use [GitHub Discussions](../../discussions):
- **Q&A** - Get help from the community
- **Ideas** - Share and discuss new ideas
- **Show and Tell** - Share your DevSpaces setup

## Issue Guidelines

Before opening an issue:

1. **Search first** - Your issue might already exist
2. **One issue per problem** - Don't combine multiple bugs
3. **Be specific** - Include versions, logs, and reproduction steps
4. **Use templates** - They help us help you faster
